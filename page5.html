Arduino Based Brushed Drone

Ardunio based drone using brushed dc motor and sensors with open-source MultiWii


Arduino
Drone
Programming
Completed: March 2021

Project Overview

This project aimed to improve self-learning and it is an introduction to robotic applications. Using Arduino and understanding drone concept was the primary goal in this project. Here, open-source Arduino drone software MultiWii is used but adjusted for brushed dc motors. All the parts used here bought from China.

Challenge

The primary challenge was understanding embedded programming and drone concept. Also hand works like soldering etc. was challenging at first as well.

PoC Process

Firstly, all parts are bought and soldered. using hot silicone, they are placed on the drone frame. Arduino is used as flight controller and since receiver mini NRF24l01 is soldered on arduino, i made transmitter is as well with arduino and NRF24l01 Pa+Lna.
