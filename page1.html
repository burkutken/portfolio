Design of an Automatic Loading System for 120 mm Tank Ammunition

Completed: June 2025

Project Overview

This is a graduation design project focuses on the development of an Automatic Loading System for 120 mm Tank Ammunition. The system is designed to operate under military conditions, with the capability to function on 30° incline platform, load ammunition within 6 seconds, and maintain a total system mass under 500kg. The system integrates a 22-round magazine, actuator-driven gripper units, and a modular platform to carry the magazine system. The goal was increase to operational efficiency, reduce human error, and ensure safety and reliability in defense scenarios, all within a strict budget of $30,000.

Challenge

In this project i was responsible with:

  • Gripper Static and Shock Analysis using ANSYS
  • Magazine System Design
  • Magazine System Components Static and Shock Analysis using ANSYS
  • Magazine Control System Design

Here, challenges were:

  • Meeting military standars (MIL-STD-810H) for shock analysis
  • Designing a compact system capable of operating on slope and in tight spaces which 22 ammunitions can fit
  • minimizing mechanical complexitiy and material selection
  • Moving target (selected) gripper within 6 seconds to releasing position

Solution

In the Magazine System Design:

  • Chain-driven system with track rollers moving V-groove rails used.
  • Aluminum 6061 is selected as a result of several pugh matrices considering weight-cost-strength.
  • ANSYS strength analysis conducted respect to MIL-STD-810H standard for shock especiall.y
  • BLDC Motor is calculated and selected for translation and simulated using MATLAB and Sİmulink.

In the magazine control system design, motor shaft position is controlled using built in encoder and system design as a close loop system. Here, additionally notes from the class MECH4480 being used since motor rotates causes translation so there is rotation-to-translation conversion. PD Controller is used to control the motor shaft position and to select the proper PD gains, damping ratio iterated and 0.8 damping ratio is selected since it has less overshoot and faster transient.

Results

The MWTS has been successfully implemented in three pilot locations:

  • System met all functional requirements with a load time 6 seconds
  • Factor of Safety (FoS) > 4.9 in worst-case shock; all components passed fatigue and vibration analysis.
  • Full compliance with MIL-STD-810H for environmental and operational conditions.
  • System is modular and compact, suitable for retrofitting into different armored vehicles.
