Gyroid Design and Testing Project

Design of several Gyroids, Printing, Testing the Strenght and Selecting Best Design using Machine Learning

Project Overview

Gyroids are complex geometries that can be found in the nature. These geometries are hard to manufacture that addative manufacturing helps to produce. In this project, Gyroids are designed using nTop program and 3D printed with SLA material. Later, they are put under compression testing and the results are collected, then, using machine learning algorithms such as linear regression, polynomial regression, decision tree and random forest regression, best design will be selected.

This project is accepted and supported by Tübitak 2209a 2024-1 program. No: 1919B012414175

Challenge

  • Gyroid structures exhibit nonlinear mechanical behavior under compression, making it difficult to predict how parameters (cell size, wall thickness, density) interact to affect performance.
  • SLA printing of intricate gyroid geometries requires strict control over resolution, material curing, and support structures.
  • Limited prior data on SLA-printed gyroids risks overfitting machine learning (ML) models.
  • Compression tests may yield inconsistent data due to material variability or testing errors.

Solution

  • Use Random Forest Regression and Polynomial Regression to handle complex parameter interactions.
  • Combine experimental data with synthetic data from literature
  • Optimize models using MAE, MSE, and R²
  • Use simulation data (FEA) if physical tests fail
